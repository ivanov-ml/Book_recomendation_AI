# Book Recommendation System

## 📌 Описание проекта

Рекомендательная система книг на основе **SVD (Singular Value Decomposition)** — метода матричной факторизации. Система анализирует историю оценок пользователей и предсказывает, какие книги могут понравиться новому пользователю.

**Бизнес-задача:** персонализированные рекомендации для увеличения вовлеченности и удержания пользователей книжного сервиса.

---

## 🎯 Техническая задача

| Параметр | Значение |
|----------|----------|
| **Вход** | Оценки пользователей (user_id, book_id, rating) |
| **Выход** | Топ-N книг с предсказанным рейтингом для пользователя |
| **Подход** | Collaborative filtering через SVD |
| **Данные** | Goodreads dataset (10k книг, 5.9M оценок, 53k пользователей) |

---

## 📊 Данные

- **Источник:** [Goodreads 10k dataset](https://www.kaggle.com/datasets/zygmunt/goodbooks-10k)
- **Книги:** 10,000 книг (названия, авторы, средний рейтинг)
- **Оценки:** 5.9 млн оценок от 53,000 пользователей
- **Плотность матрицы:** 1.12% (типично для рекомендательных систем)
